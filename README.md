# Attacker Lure: Honeypot & Automated Threat Intelligence Pipeline

 Project OverviewThis project implements a low-interaction honeypot to attract and log malicious network activity, coupled with an automated pipeline to process, enrich, and visualize the captured Threat Intelligence (TI) data in real-time.The core goal is to gain hands-on experience in deception technology, log analysis, and Security Orchestration, Automation, and Response (SOAR)-like processes, providing actionable insights into attacker Tactics, Techniques, and Procedures (TTPs).
